# :white_flower: ModBus :white_flower:

Script take IP as input and brute force against all the registers. Identified valid registered will be exported to excel file

## Pre-requisite:
* pip install -r requirements.txt


## Scan:
* Coil Register
* Holding Register
* Discrete Input
* Input Register

## Execution:
* python main.py -i ip

## Output
* Excel File
